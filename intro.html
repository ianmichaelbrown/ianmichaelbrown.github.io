The journey begins...

November 29, 2020 Ian

So, I finally got around to doing what I knew I had to do if I was ever going to be a better developer - I've started learning properly about web development. I already know a bit about this, but have a fair bit more experience in the non-web world (what used  to be called 'desktop' applications), so now it's time to get good at what are now  called desktop applications and ultimately bring everything together to turn myself into a fully-fledged FULL STACK DEVELOPER!

This journey is all about learning and there are many, many things to be learnt, but here is what I've discovered so far in order to get this site going:

  • HTML Templates:  These provide a starting point for quickly creating a site with a certain style.  There are many sources of such templates.  This site uses one called 'Future Imperfect' from HTML5 UP!, which has been adapted to fit with the style of this site.
  • GitHub Pages:  One of many places where you can create a new website quickly and easily.  Go to the main page to learn how to create a site.
  • Git:  I've used Git for a while but not really from the command line, so I'm building my knowledge on this.  I'm also learning about and using Windows Terminal and Powershell to work with Git commands.  To help with this, I used some great info found in a blog post by the wonderful Scott Hanselman, showing a really nice way to decorate your terminal prompt to make things clearer when working with the Git command line.
  • Font Awesome:  The social icons at the bottom of this site's pages come from Font Awesome, which has all the icons you could ever need!
  • Images:  When developing a site it's really useful to be able to add placeholder images, until you've found or created the real ones.  Unsplash.com is a hugely-useful resource for this purpose.
  • HTML & CSS:  I've used these for a while but only on a fairly basic level, so my aim is to become really proficient in both, as this is essential for being a web developer.  Examining and updating an HTML template, such as the one this site is based on, has already proved to be a great way to increase my knowledge in both these areas, but this will really improve during the process of creating my personal website.  I've already watched or read many  tutorials on these and a lot of other 'webby' topics, but the generally-accepted way to avoid tutorial hell/purgatory is to actually try things out rather than just watch hundreds of tutorials.  One of many good examples of what to do is in this video by the ever-helpful Joshua Fluke,  whose YouTube channel has a whole heap of useful and interesting content.
