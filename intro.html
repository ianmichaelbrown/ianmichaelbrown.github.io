How I got started and where I am now

Things I've learned about so far in order to get this site going:

  • HTML Templates: this site is based on a template from HTML5 UP!  There are many other sources of such templates and they all provide a starting point for quickly creating a site with a certain style.  This site is an adaptation of the 'Future Imperfect' template.
  • GitHub Pages: One of many places where you can create a new website quickly and easily. Go to the main page to learn how to create a site.
  • Git: I've used Git for a while but not really from the command line, so I'm building my knowledge on this.  I'm also using Windows Terminal and Powershell  to work with Git commands. I came across a really nice way to decorate your terminal prompt to make things clearer when working with Git.  All the info you should need can be found in a blog post by the wonderful Scott Hanselman.
  • Font Awesome: The social icons at the bottom of this site's pages come from Font Awesome, which has all the icons you could ever need!
  • Images: When developing a site it's really useful to be able to add placeholder images, until you've found or created the real ones.  Unsplash.com is a hugely-useful resource for this purpose.
  • HTML & CSS: I've used these for a while but only on a fairly basic level, so my aim is to become really proficient in both, as this is essential for being a web developer.  Examining and updating an HTML template, such as the one this site is based on, has proved to be a good way to increase my knowledge in both these areas, but this will really improve during the process of creating my personal website.  I've already watched or read many tutorials on these and a lot of other 'webby' topics, but the generally-accepted way to avoid tutorial hell/purgatory is to actually try things out  rather than just watch hundreds of tutorials on web development. One of many good examples of what to do is in this video by the ever-helpful Joshua Fluke,  YouTube channel has a whole heap of useful and interesting content.
